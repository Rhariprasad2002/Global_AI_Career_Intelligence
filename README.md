# Global_AI_Career_Intelligence
An intelligent and interactive AI Career Intelligence System built using Streamlit, designed to predict job salaries in the AI and Data Science domain.
The application leverages a Gradient Boosting Machine Learning model, with feature scaling and label encoding to ensure accurate predictions.
Users can input key job details such as job title, experience level, company size, location, education, and required skills, and receive an estimated annual salary in USD.
